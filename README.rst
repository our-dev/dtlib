=================
dtlib
=================

.. contents:: 目录




项目说明
=============

- 开发过程中一些常用的工具函数集合

运行环境
=============

- Python 3.5



概述
=============

用于个人在开发阶段的快速日志成型。其它项目可以直接引用此库，进行简单的封装，可以做可插拔，这样做主要是避免繁杂的配置。

使用方法：

- 安装库
- 在项目中编写简单的封装函数，传入字符串




安装方法
=============

.. code::

    sudo python setup.py install
